# STRATEGIC NOTES

## Why This Program Fits PergoLuxx:

1. **0% Interest Loan** - Dramatically reduces carrying costs vs. conventional construction financing

2. **75% Rural Set-Aside** - Less competition if you target rural counties

3. **Construction Expertise** - Your existing contractor license and experience qualifies you

4. **I.H.P. Integration** - Install smart home monitoring in all units as a differentiator (protects your investment, attracts tenants, potential insurance discounts)

5. **Build Portfolio** - Creates rental income stream while building development track record

## Recommended Strategy:

1. Start small (5-10 units) to prove capability
2. Target rural preference county for higher approval odds
3. Use I.H.P. integration as unique selling point
4. Build relationship with OHFA staff before formal application
5. Consider partnering with experienced multifamily developer for first project
